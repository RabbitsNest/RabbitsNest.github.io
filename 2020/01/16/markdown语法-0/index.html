斜体和粗体

语法说明:

  1. 加粗
    : 要加粗的文字左右用两个 ““ 号包裹
    如: 加粗文字
    code: ` *加粗文字     `
  2. 斜体
    : 要倾斜的文字左右用一个 ““ 号包裹
    如:     倾斜文字
    code: `     倾斜文字* `
  3. 斜体加粗
    : 要倾斜和加粗的文字左右用三个 ““ 号包裹
    如: **    加粗倾斜文字
    code: `     加粗倾斜文字* `
  4. 删除线
    ： 要加删除线的文字左右用两个 ~ 号包裹
    如: 加删除线文字
    code: ~~加删除线文字~~
